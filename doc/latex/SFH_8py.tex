The file contains the functions to determine the star formation rate. The function representative\+\_\+\+SFH determines an appropriate value for the star formation rate at a given age. It allows for an optional additional time delay, due to a delay in formation of the binary, or if time is required to move to the correct frequency bin. The functions SFH\+\_\+\+MD, SFH2, SFH3, and SFH4 are star formation histories that can be selected. \begin{DoxyAuthor}{Author}
